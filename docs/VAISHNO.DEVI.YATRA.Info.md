# VAISHNO DEVI YATRA -- COMPLETE RUNBOOK

## Overview

-   Base: Katra (J&K)
-   Shrine: Bhawan (Holy Cave)
-   Optional: Bhairavnath Temple
-   Total Round Trip Distance: \~26--28 km
-   Best Start Time: 4:30--5:30 AM

------------------------------------------------------------------------

# ROUTE OPTIONS

## 1. Traditional Track (Classic Route)

Katra → Banganga → Charan Paduka → Adhkuwari → Sanjichhat → Bhawan

## 2. New Himkoti Track (Recommended for Families)

Katra → Adhkuwari (Old Route) → Himkoti → Sanjichhat → Bhawan - Wider,
cleaner - Less pony traffic - Easier gradient

## 3. Assisted Options

-   Pony / Palki: Available from Katra
-   Battery Car: Adhkuwari → Sanjichhat → near Bhawan
-   Helicopter: Katra → Sanjichhat
-   Ropeway: Bhawan → Bhairavnath

------------------------------------------------------------------------

# LANDMARK-BY-LANDMARK DETAILS

## 1. KATRA (Base Camp)

Altitude: \~2500 ft\
Facilities: Registration, cloak room, food, pony booking\
Start Point: Darshani Deodhi

------------------------------------------------------------------------

## 2. DARSHANI DEODHI (1 km from Katra)

Symbolic start of trek\
Security check\
First view of Trikuta Hills

Time from Katra: 15--20 mins walk

------------------------------------------------------------------------

## 3. BANGANGA (2.5 km from Katra)

Altitude: \~2800 ft\
Significance: Sacred stream created by Mata\
Facilities: Rest stops, food, toilets

Time from Katra: 45--60 mins

------------------------------------------------------------------------

## 4. CHARAN PADUKA (\~5 km total)

Significance: Footprints of Mata\
Terrain: Moderate uphill

Time from Banganga: 1--1.5 hrs

------------------------------------------------------------------------

## 5. ADHKUWARI (\~9 km total)

Significance: Garbh Joon cave (meditation site)\
Major rest complex\
Battery car boarding point

Time from Charan Paduka: 2--2.5 hrs

------------------------------------------------------------------------

## 6. HIMKOTI (New Track Only)

Scenic viewpoint\
Less crowded\
Cafeteria & rest area

Located between Adhkuwari and Sanjichhat

------------------------------------------------------------------------

## 7. SANJICHHAT (\~11.5 km total)

Highest point (\~6200 ft)\
Helicopter drop location\
Open rest plateau

Time from Adhkuwari: 1--1.5 hrs

------------------------------------------------------------------------

## 8. BHAWAN (\~13 km total)

Altitude: \~5200 ft\
Main shrine complex\
Rooms, lockers, blanket store\
Bhojanalaya available

Time from Sanjichhat: 45--60 mins

------------------------------------------------------------------------

## 9. BHAIRAVNATH (2 km from Bhawan)

Yatra incomplete without visit\
Steep climb\
Ropeway available

Time: 45--60 mins walk or 5 mins ropeway

------------------------------------------------------------------------

# DESCENT OPTIONS

## Option A: Traditional Return

Bhawan → Adhkuwari → Banganga → Katra

## Option B: New Track Return (Recommended)

Bhawan → Himkoti → Adhkuwari bypass → Katra

Descent Time: 4--5 hrs

------------------------------------------------------------------------

# TIME ESTIMATES (FAMILY PACE)

Katra → Adhkuwari: 4--5 hrs\
Adhkuwari → Bhawan: 2--3 hrs\
Bhairav round trip: 1.5--2 hrs\
Full descent: 4--5 hrs

------------------------------------------------------------------------

# PRACTICAL TIPS

-   Generate Yatra Parchi 1 day before trek
-   Book Bhawan accommodation 60 days prior
-   Start trek early to avoid heat and rush
-   Carry light woolens (cold at Bhawan)
-   Hydrate frequently
-   Senior citizens: Use battery car + ropeway
