# Development Guide

```bash

npm run dev

npm run preview
```

## Project Architecture

### Core Modules

#### 1. **map.js** - Map Management
- Initializes Leaflet map
- Manages tile providers (OSM primary, MapTiler fallback)
- Handles tile loading errors with automatic fallback
- Provides map control methods

#### 2. **gps.js** - GPS Tracking
- Geolocation API wrapper
- Battery saver mode (reduced accuracy)
- Auto-center functionality
- Distance calculations (Haversine formula)
- User position marker and accuracy circle

#### 3. **layers.js** - Route Layers
- Manages multiple route layers
- Milestone marker creation and styling
- Layer visibility toggling
- Bounds fitting

#### 4. **storage.js** - Data Persistence
- LocalStorage for settings and state
- IndexedDB for notes and photos (future)
- Route state management
- Visited milestones tracking

#### 5. **app.js** - Main Application
- Orchestrates all modules
- Route loading and initialization
- UI event handling
- Progress tracking

## Adding a New Route

### Step 1: Create Route Directory

```bash
mkdir -p public/routes/your-route-name
```

### Step 2: Create config.json

```json
{
  "id": "your-route-name",
  "name": "Your Route Display Name",
  "description": "Route description",
  "center": {
    "lat": 0.0,
    "lng": 0.0,
    "zoom": 13
  },
  "routes": [
    {
      "id": "main-route",
      "name": "Main Route",
      "file": "route.geojson",
      "color": "#2196F3",
      "visible": true,
      "weight": 4
    }
  ],
  "milestones": {
    "file": "milestones.geojson",
    "auto_mark_distance": 30
  }
}
```

### Step 3: Create route.geojson

```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "id": "main-route",
        "name": "Main Route",
        "difficulty": "moderate",
        "distance": 10.0
      },
      "geometry": {
        "type": "LineString",
        "coordinates": [
          [longitude, latitude],
          [longitude, latitude]
        ]
      }
    }
  ]
}
```

### Step 4: Create milestones.geojson

```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "id": 1,
        "name": "Checkpoint 1",
        "elevation": 1000,
        "type": "checkpoint",
        "distance_from_start": 0
      },
      "geometry": {
        "type": "Point",
        "coordinates": [longitude, latitude]
      }
    }
  ]
}
```

### Step 5: Update app.js

Change the route ID in `app.js`:

```javascript
await this.loadRoute('your-route-name');
```

## GPS Coordinates

### Finding Coordinates

1. **Google Maps**: Right-click → "What's here?" → Copy coordinates
2. **OpenStreetMap**: Click location → See coordinates in URL
3. **GPS Device**: Export GPX and convert to GeoJSON

### Coordinate Format

- GeoJSON uses `[longitude, latitude]` (NOT lat, lng!)
- Decimal degrees format: `[74.93333, 32.98944]`
- Precision: 5-6 decimal places is sufficient

### Tools

- **geojson.io**: Visual GeoJSON editor
- **GPX to GeoJSON**: https://mapbox.github.io/togeojson/
- **Distance Calculator**: https://www.movable-type.co.uk/scripts/latlong.html

## Troubleshooting

### Icons Not Showing

Generate icons using `scripts/generate-icons.html` and place in `public/icons/`

### Map Not Loading

Check browser console for tile loading errors. Ensure internet connection for first load.

### GPS Not Working

- Ensure HTTPS (required for geolocation)
- Check browser permissions
- Test on actual device (not always available in desktop browsers)

### Build Errors

```bash
# Clear cache and reinstall
rm -rf node_modules package-lock.json
npm install
npm run build
```
